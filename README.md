  ##  🎨 AI Image Studio

An AI-powered Image Generation app built using **Streamlit** and **Pollinations AI** as part of the **MirAI School of Technology – Virtual Summer Internship 2026 (AI Builder Track)**.

## 🚀 Features

- 🎨 Generate AI images from text prompts
- 📏 Adjustable image width and height
- 🖌️ Multiple art styles
  - Realistic
  - Anime
  - Cyberpunk
  - Fantasy
  - Digital Art
- ✨ Magic Enhance mode for higher-quality prompts
- 🎲 Surprise Me! random prompt generator
- 📥 Download generated images as `.png`
- ⚡ Simple and responsive Streamlit interface

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Requests
- Pollinations AI

## 📦 Installation

Clone the repository:

```bash
git clone <your-repository-link>
cd <repository-folder>
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
